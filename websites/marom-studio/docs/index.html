Accessibility Statement

MAROM Digital Studio is committed to ensuring digital accessibility for people with disabilities. We strive to meet WCAG 2.0 Level AA standards and Israeli Standard IS 5568, in compliance with the Equal Rights for Persons with Disabilities Law (1998) and Accessibility Regulations (2013).

Accessibility features on this site:

  • Full keyboard navigation support
  • Screen reader compatible structure with ARIA landmarks
  • Accessible color contrast ratios (minimum 4.5:1)
  • Skip-to-content link for keyboard users
  • Comprehensive accessibility widget with 17 adjustment options
  • Reduced motion support for users with vestibular disorders
  • SVG-based accessibility icon
