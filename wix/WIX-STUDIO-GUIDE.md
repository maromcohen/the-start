# 🍸 תכנית בנייה מלאה ל-Wix Studio — Cocktails & Dreams

מדריך צעד-אחר-צעד לשחזור האתר הקולנועי **נייטיב** ב-Wix Studio.
כל ההחלטות כבר מוכנות — צבעים, טקסטים, פונטים, אנימציות. רק לפתוח ולבנות.

> **למה Wix Studio ולא העורך הרגיל?** רק ל-Studio יש אנימציות גלילה
> (Reveal / Parallax), וידאו ברקע של סקשן, ושליטה מלאה ברספונסיביות —
> בדיוק מה שצריך לאתר קולנועי.

## 0. הכנה — לפני שמתחילים

### א. פתיחת Wix Studio
1. נכנסים ל-**wix.com** → להמיר/לפתוח את הפרויקט ב-**Wix Studio** (לא העורך הקלאסי).
2. מתחילים מ-**Blank Canvas** (קנבס ריק) כדי לקבל שליטה מלאה.

### ב. הגדרת שפה ו-RTL
- האתר **עברית מימין לשמאל**. ב-Studio: Settings → Language → עברית, Text direction: RTL.

### ג. העלאת המדיה
מעלים ל-Media Manager את כל הקבצים (כבר קיימים אצלך ב-Wix):
| קובץ | שימוש |
|------|-------|
| לוגו עגול | ניווט + פוטר |
| וידאו hero | רקע מסך פתיחה |
| וידאו מזיגה | סקשן The Pour Studio |
| תמונת "צ'irs" (כוסות) | סקשן אודות |
| תמונת אפרול שפריץ | כרטיס קוקטייל |
| תמונת פאשן סמאש | כרטיס קוקטייל |

- כפתורים: מסגרת זהב, פינות מעוגלות מלאות (גלולה), במעבר עכבר מתמלאים זהב.

## 2. 🧭 סרגל ניווט (Header)

**סוג:** Header קבוע (Fixed/Sticky), שקוף בהתחלה.

- **צד ימין (RTL):** לוגו עגול (38px) + טקסט "COCKTAILS & DREAMS" וקטן מתחת "MOBILE BAR" בזהב.
- **מרכז:** קישורים — `אודות · תפריט · אירועים · גלריה · הזמנה` (כל אחד עוגן לסקשן).
- **צד שמאל:** כפתור "דברו איתנו →" (מסגרת זהב).

**אנימציה:** Scroll Effect → כשגוללים מטה, הרקע הופך לכהה-שקוף עם טשטוש (Glass blur). ב-Studio: Header → Scroll → "Change background on scroll".

---

## 3. 🎬 מסך פתיחה (Hero) — הסקשן הקולנועי

**סוג:** סקשן בגובה מסך מלא (100vh).

1. **רקע:** וידאו ה-hero על מילוי הסקשן (Section Background → Video). הגדרות: Autoplay, Muted, Loop.
2. **שכבת כהות מעל הוידאו:** Overlay גרדיאנט — שקוף למעלה, כהה (#0A0708) למטה. נותן קריאוּת לטקסט.
3. **תוכן ממורכז:**
   - תווית קטנה: `MOBILE BAR SERVICE` (Marcellus, זהב, ריווח רחב)
   - כותרת ענקית: **Cocktails & Dreams** (ה-`&` בנטוי ורוד)
   - שורת תיאור: "חוויית קוקטיילים **פרימיום** לכל אירוע" (המילה "פרימיום" בזהב בהיר)
4. **רמז גלילה למטה:** קו דק עם נקודה שיורדת + הכיתוב `SCROLL`.

**אנימציות (Studio → Animations):**
- כותרת ותווית: **Reveal — Fade up** עם השהיה מדורגת (0.2s בין כל אלמנט).
- רמז הגלילה: אנימציית לופ עדינה (נקודה יורדת).

## 4. ➡️ רצועת מילים נעה (Marquee)

טקסט שרץ אופקית בלולאה אינסופית, עם כוכב זהב ✦ בין כל מילה:
> קוקטיילים ✦ חתונות ✦ אירועי חברה ✦ בר מובייל ✦ ימי הולדת ✦ (חוזר)

הפונט: Cormorant Garamond נטוי, גדול, שקיפות 50%.
ב-Studio: אפשר עם **Marquee/Scrolling text** מובנה, או רצועה עם אנימציית Loop אופקית.
